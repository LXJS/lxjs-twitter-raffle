# LXJS Twitter Raffle

## Install

```bash
$ npm install -g lxjs-twitter-raffle
```

## Create a credentials file

Named `.twitter_credentials.json` inside your home dir:

```javascript
{
	"consumer_key": "xxx",
	"consumer_secret": "xxx",
	"access_token_key": "xxx",
	"access_token_secret": "xxx"
}
```

## Run

```bash
$ twitter-raffle <query>
```
